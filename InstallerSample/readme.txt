How to use this Installer Sample.

This Installer Sample is just a directory structure for the .iss script file to generate an installation file. The installation executable will be created in the "INSTALLER" folder when the script is run in INNO SETUP.

Directions on how to use the .iss script file in the file's comments. Run the script in INNO SETUP, which is program located on all lab computers.

Use the installation executable that is created in the "INSTALLER" folder in your Final submission .zip file.
